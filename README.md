# Sales Data Analysis Using Python 📊

## Overview
This project focuses on performing end-to-end sales data analysis using Python.

The analysis includes:
- Data Cleaning
- Missing Value Handling
- Outlier Detection
- Exploratory Data Analysis (EDA)
- Revenue Analysis
- Customer Analysis
- Sales Trend Visualization

The project helps understand customer behavior, sales performance, product trends, and business insights using real-world sales data.

---

## 🚀 Features

- Data preprocessing and cleaning
- Missing value treatment
- Duplicate data checking
- Outlier removal using IQR
- Revenue and profit analysis
- Customer segmentation
- Repeat customer analysis
- Monthly sales trend analysis
- City-wise revenue analysis
- Category performance analysis
- Interactive visualizations using Matplotlib & Seaborn
